Задание 1.
1. Есть ли среди приведенных определений недопустимые?
a) int *ip;
b) char *s = 0;
c) int i; 
double* dp = i;
d) int *ip, *ip2;
e) float *pf = Null;

Ответ: Есть, под буквой b, c, e.
	b) Указатель указывает на адрес, поэтому при инициализации нужно указывать на 
	адрес переменной, а не на число.
	с) Указатель может указывать на адрес переменной только этого же типа.
	e) У NULL тип - unsigned long int, а у *pf - float, из-за разницы в
	типах указатель не будет работать так, как надо.

2. Что выполняет следующий код:
int i = 42, j = 1024;	//Инициализиция переменных i и j
int *p1 = &i, *p2 = &j	//Инициализация указателей *p1 = 42, *p2 = 1024
*p2 = *p1 * *p2;	//*p2 = 43008
*p1 *= *p1;		//*p1 = 1764

Коментрии оставлены в самом коде, в виде "//...".
