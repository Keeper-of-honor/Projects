Качаешь git, если он не скачан:
sudo apt install git

Делаешь в любой папке:
git clone https://github.com/prog-kkkmt/p1-18.git
И там появится папка репрезитория.
Чтоб перейти на свою ветку делаешь
git checkout <имя ветки>

А когда захочешь отослать изменения сделанные на компе в гитхаб,
то надо перейти внутрь каталога репоитория
cd p1-18
и пишешь в корне репозитория
git add .
git commit -m "твое описание изменений"
git push
Потом попросят имя и пароль.
